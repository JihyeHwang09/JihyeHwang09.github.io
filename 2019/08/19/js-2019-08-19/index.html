JavaScript- null VS falsy 예외 처리

2019-08-19
⏱ 읽기 소요 시간: 3 분

null에 대한 예외 처리

  • null에 대해서만 예외 처리를 한다.

falsy에 대한 예외 처리

  • undefined, 빈문자열(''), 숫자 0등 falsy에 다 예외처리한다.

falsy란?

  • boolean 문맥에서 false로 평가되는 값을 의미한다.

  • falsy의 종류

    • false
    • null
    • undefined
    • 0
    • NaN
    • ''
